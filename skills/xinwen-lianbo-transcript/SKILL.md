# 新闻联播文字稿获取

## 整体流程

```
日期列表接口                    分段视频页面                      完整逐字稿
tv.cctv.com/lm/xwlb/day/  →  遍历 [视频] 链接               →  提取 #content_area
{YYYYMMDD}.shtml           每条含完整逐字稿                    按顺序拼接
```

## 获取指定日期的栏目列表

### 数据接口
```
https://tv.cctv.com/lm/xwlb/day/{YYYYMMDD}.shtml
```
返回 HTML 片段，包含当日所有视频条目：
- **完整版**（标记 `sql0`）：仅有标题摘要，**无逐字稿**
- **分段视频**（标记 `sql1`）：含 **完整逐字稿**

### 栏目交互页
```
https://tv.cctv.com/lm/xwlb/
```
内置日历选择器，支持 2011 年至今的历史日期。

## 获取完整逐字稿

### 分段视频正文URL
```
https://tv.cctv.com/{YYYY}/{MM}/{DD}/VIDE{RANDOM}{YYMMDD}.shtml
```

### 文字稿位置
- **text 模式**：页面"主要内容"区域，以 `央视网消息（新闻联播）：` 开头，到 `编辑：` 结束
- **html 模式**：容器 `#content_area`

完整版页面不含逐字稿，需遍历所有分段视频页面逐一提取后按顺序拼接。国内联播快讯和国际联播快讯的分段页面已包含多条新闻。

### 日期支持范围
- 2021年11月1日至今：`//tv.cctv.com/lm/xwlb/day/{YYYYMMDD}.shtml`
- 2013年7月9日-2021年10月31日：`http://cctv.cntv.cn/lm/xinwenlianbo/{YYYYMMDD}.shtml`
- 2011年4月6日-2013年7月8日：同上

## 注意事项
- 周末无节目时数据自动回退到前一天
- 当天节目 19:10 之后才可查看，在此之前默认显示前一天
- 无需登录
- 建议 text 格式取正文，html 格式解析链接
